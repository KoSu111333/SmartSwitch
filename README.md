# SmartSwitch Project

## 1. 프로젝트 개요

블루투스 통신을 통한 원격 제어가 가능한 스마트 스위치를 개발

*   대상: 전등 스위치, 도어락 스위치
*   목표: 사용자의 편의성 및 주거 환경 개선

## 2. 시스템 구조 및 하드웨어 사양

### 클럭(CLOCK) 설정

*   SYSCLK: 64 MHz
*   HCLK: 64 MHz
*   PCLK1: 64 MHz
*   PCLK2: 32 MHz

### 3. 주요 핀 할당

*   `PA2` : 디버깅 UART TX 핀
*   `PA3` : 디버깅 UART RX 핀
*   `PA8` : 서보 모터 PWM 출력 핀
